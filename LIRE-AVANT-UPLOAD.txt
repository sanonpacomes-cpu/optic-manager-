IMPORTANT - NE PAS UPLOADER LES FICHIERS API À LA RACINE

Après décompression, il faut uploader tout le contenu en gardant le dossier api.

Sur GitHub, à la racine, tu dois voir :
- api
- database
- index.html
- main.jsx
- style.css
- package.json
- vercel.json

Dans le dossier api, tu dois voir :
- health.js
- login.js
- ensure.js
- setup.js
- sales.js
- users.js
- clients.js
- agencies.js
- db.js
- security.js
- generate-login-id.js

Si health.js est à la racine au lieu d'être dans api, /api/health donnera encore 404.
